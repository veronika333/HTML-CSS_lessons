Input types:
button Needs value attribute
checkbox It is a checkbox Needs a few different value attribute (to have multiple variants)
color The value is a color
date Date of birth: Needs a name
datetime-local The meeting starts (date, time): Needs name
e-mail Can be "multiple"
file If multiple, can allow multiple files
hidden First name: Defines a hidden input field, should have a value
image The input type="image" sends the X and Y coordinates of the click that activated the image button.
month Passport is valid until: Define a month and year control (no time zone)
number Define a field for entering a number (You can also set restrictions on what numbers are accepted)
password Define a password field (characters are masked), can set ex: maxlegth="7"

range Define a range control (like a slider control)

search Search Google: Defines a text field for entering a search string
submit Define a submit button
tel Telephone: Define a field for entering a telephone number
text Your name: Define a single-line text field that a user can enter text into
time Select a time: Define a control for entering a time (no time zone)
url Add your website: Define a field for entering a URL
week Define a week and year control (no time zone)
select Can select a few options
textarea You can drag text area and make it bigger
